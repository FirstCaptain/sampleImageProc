# sampleImageProc


Just a quick sample of segmenting out and object and the use of contours. 

### Dependencies

Assumes you have cmake, opencv other prereq's per the cmake file installed. 

### Build and Run 
Clone then repo

~~~
mkdir build
cd build
cmake ..
make
./sample ../key.jpg
~~~
